# Применение фильтра размытия на изображение с использованием CUDA
## Описание алгоритма
- Разделение изображения на RGB каналы;
- Обработка каждого канала на GPU:
  1. Создается shared-память размером block_size*block_size пикселей для каждого блока;
  2. Каждый блок потоков загружает фрагмент изображения в shared-память;
  3. Потоки обрабатывают тайлы размером filter_size*filter_size пикселей.

  Алгоритм размытия в ядре CUDA:
    - Для каждого пикселя рассматривается область filter_size*filter_size;
    - Усредняем значения пикселей в этой области;
    - Результат записываем в `out`.

- Обработанные каналы объединяются в итоговое изображение;
- Освобождаем 
## Функции
- `CUDA_CHECK_RETURN` - макрос для проверки ошибок CUDA
- `processImg()` - основное ядро CUDA для обработки изображения:
  - Загружает данные в shared memory;
  - Применяет фильтр усреднения.
- `void processImageChannel()` - управляет обработкой канала:
  - Выделяет память на устройстве CUDA;
  - Копирует данные между хостом и устройством;
  - Запускает CUDA ядро;
  - Освобождает ресурсы.
- `applyBlurFilter() - то, что раньше было в мейне`:
  - Замеряет время;
  - Разделяет изображение на каналы;
  - Вызывает processImageChannel для каждого канала;
  - Записывает результат в output.
- `main()`
  - Загружает изображение;
  - Вызывает applyBlurFilter для разных параметров;
  - Выводит время в консоль.

## Требования
- Библиотека OpenCV;
- CUDA;
- Изображение `test_image.png` в директории проекта.
