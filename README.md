# Применение фильтра размытия на изображение с использованием CUDA
## Функции
- `CUDA_CHECK_RETURN` - макрос для проверки ошибок CUDA
- `processImg()` - основное ядро CUDA для обработки изображения:
  - Загружает данные в shared memory;
  - Применяет фильтр усреднения.
- `void processImageChannel()` - управляет обработкой одного цветового канала:
  - Выделяет память на устройстве CUDA;
  - Копирует данные между хостом и устройством;
  - Запускает CUDA ядро;
  - Освобождает ресурсы.
- `main()`
  - Загружает изображение и разделяет на каналы;
  - Обрабатывает каждый;
  - Объединяет результат и сохраняет изображение.

## Требования
- Библиотека OpenCV;
- CUDA;
- Изображение `test_image.png` в директории проекта.
